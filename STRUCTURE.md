## 🎨 Animation Flow

```
Page Load (Hero):
├─ 0.0s: Name fades in + slides up
├─ 0.2s: Role text appears
├─ 0.4s: Description fades in
├─ 0.6s: Buttons slide up
└─ 0.8s: Social icons appear

Scroll (Each Section):
├─ User scrolls to section
├─ Section enters viewport (20% visible)
├─ Trigger animation
└─ Content animates (fade-up/slide)

Project Cards Hover:
├─ Mouse enters card
├─ Scale: 1.0 → 1.05 (300ms)
├─ Shadow: normal → red glow
├─ Translate: 0 → -10px
└─ Image: scale 1.1

Click Card → Modal:
├─ Modal backdrop fades in
├─ Content scales 0.9 → 1.0
├─ Shows full project details
└─ Close: reverse animation
```
